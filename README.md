## Voraussetzungen
Anaconda-Environment mit python und den Paketen pymunk und pygame. <br>
Evtl. mit pip install in der Powershell installieren.

## Start 
1. Anaconda starten
2. In das pymonk-Environment wechseln
3. VSCode aus Anaconda heraus starten
4. Projektordner öffnen
5. Gewünschtes *.py öffnen, editieren oder mit CODEX überarbeiten
6. *.py starten
